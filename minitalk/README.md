## 프로젝트 개요
UNIX signal을 이용한 소규모 데이터 교환 프로그램을 만드는 프로젝트이다.  
SIGUSR1, SIGUSR2 를 0, 1 이진수로 활용하여 데이터를 전달했다.  

비고: 수신 확인 시스템과 유니코드를 지원하게 하는 것이 보너스이다. TCP/IP의 핸드셰이크를 모방한 수신확인 시스템을 만드려 했으나 잘 안되서 보너스를 포기했다. 무슨 이유인지는 기억이 나지 않는다.
